点击这里访问百度

详解

!DOCTYPE html>

  • 这是 文档类型声明，告诉浏览器我们使用的是 HTML5 版本。
  • 每个 HTML 文件的第一行都应该有它。

<html lang="zh-CN">

  • 表示整个 HTML 文档开始了。
  • lang="zh-CN" 告诉浏览器这个页面是中文（中国地区）。

<head>...</head>

  • <head> 标签内放的是页面的元信息，不会直接显示在网页上。

  • 常见的内容包括字符编码、标题、CSS样式、JS脚本等。

  • <meta charset="UTF-8">

  • 告诉浏览器使用 UTF-8 编码方式显示网页，支持中文。

-<meta name="viewport" content="width=device-width, initial-scale=1.0">

  • viewport（视口）是浏览器用来显示网页内容的“可视区域”。
  • 在手机上，默认的视口宽度可能比屏幕要宽，所以网页看起来会被缩小。
  • 这个标签的作用是告诉浏览器：“请用设备的屏幕宽度作为页面宽度来显示网页”。
  • width=device-width → 视口宽度等于设备屏幕宽度
    initial-scale=1.0 → 初始缩放比例为 1（即不放大也不缩小）

<title>我的第一个网页</title>

  • 设置浏览器标签页上显示的标题，不会出现在网页正文中。

<body>...</body>

  • <body> 标签是页面的“身体”，真正展示给用户看的内容都放在这里。

<h1>你好，世界！</h1>

  • <h1> 是一级标题，用于强调重要内容，字体最大。
  • HTML 有 <h1> 到 <h6> 六级标题，数字越大字体越小。

<p>这是我的第一个网页。我正在学习 HTML。</p>

  • <p> 是段落标签，用来写正文内容，每个段落会自动换行。

<a href="https://www.baidu.com" target="_blank">点击这里访问百度</a>

  • <a> 是超链接标签，用于跳转到其他页面或网站。
  • href="..." 指定跳转的地址。
  • target="_blank" 表示在新标签页打开链接。

img

eg:<img src="https://via.placeholder.com/300x200" alt="示例图" width="300" height="200">

属性 含义
src 图片的地址（source）
alt 当图片无法加载时显示的文字（也对盲人读屏器有用）
width 设置图片宽度（可以写像素 px 或百分比）
height 设置图片高度，可以写“auto”自适应
title 鼠标移到图片上时显示的提示文字
style 用CSS设置图片大小、圆角、阴影等样式

常见标签和属性

标签 用途 简单示例
<p> 段落文本 <p>这是段落。</p>
<h1> ~ <h6> 标题（从大到小） <h1>主标题</h1>
<button> 按钮 <button>点击我</button>
<input> 表单输入 <input type="text">
<div> 容器、布局盒子 <div>内容</div>
<img> 图片 <img src="cat.jpg" alt="猫">
<iframe> 内嵌网页或视频 <iframe src="video.html"></iframe>
<a> 超链接，target=”_blank”表示新窗口打开 <a href="https://example.com">跳转</a>

可以嵌套，如

<a href="https://dun51.github.io" target="_blank"><button type="button">普通按钮</button> </a>

效果为点击按钮后跳转。

结构标签

  1. <header>：页面头部，一般包含网站名称、导航、搜索框等
  2. <footer>：页面底部，通常包括版权信息、联系信息等。
  3. <nav>：导航栏，通常包含网站的各个页面链接。
  4. <section>：页面区域，用来定义页面的不同区域，具有语义化作用。例如：文章内容、产品介绍等。
  5. <article>：文章内容，用于独立的内容部分，通常是新闻、博客文章等。它通常有自己的标题和内容，可以独立呈现。

文本格式化标签

  1. <b>：加粗

  2. <i>：斜体

  3. <u>：下划线

  4. <strong>：强调，通常为加粗

  5. <em>：强调，通常为斜体

  6. <br>：换行

  7. <hr>：水平线

  8. <span>：内联元素，用于标记一小段文字，可以用来设置样式。

    如 <p>这是<span style="color:red;">红色</span>文字。</p>

    这是红色文字。

  2. <ol>：有序列表
  3. <li>：列表项

表单标签

  1. 基本结构

    表单内容（输入框、按钮等）

    action：提交数据到哪个服务器地址（可以是后端处理地址）

    method：提交方式（常用 GET 或 POST）

    方式 特点
    GET 数据附在 URL 后面，适合搜索、查询
    POST 数据放在请求体中，更安全，适合登录、提交

  2. 表单内容

    • 文本输入框：<input type="text">

      name：提交给服务器的字段名

      placeholder：输入提示文字

      id + <label for="...">：点击文字可以聚焦输入框

    • 密码输入框：<input type="password">

    • 单选按钮：<input type="radio">

      同一个 name 表示一组选项，只能选一个。

    • 多选框：<input type="checkbox">

      多个选项可同时选中。

    • 下拉菜单：<select>

    • 文本区域（多行输入）：<textarea>

    • 按钮：<button> / input
